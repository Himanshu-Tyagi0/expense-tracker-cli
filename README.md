# Expense Tracker CLI

A command-line based Expense Tracker application built using Python.  
This project allows users to record, manage, and analyze their expenses using a simple terminal interface.

---

## Features

- Add new expenses
- View all recorded expenses
- Monthly expense summary
- Category-wise spending analysis
- Persistent storage using CSV file

---

## Tech Stack

- Python
- CSV module
- Datetime module

## How to Run

1. Clone the repository:

   git clone https://github.com/Himanshu-Tyagi0/expense-tracker-cli.git

2. Navigate into the project folder:

   cd expense-tracker-cli

3. Run the application:

   python expense_tracker.py
